Class ExtensibleGamepad

  • java.lang.Object
    • org.ftccommunity.networkedopmode.robot.ExtensibleGamepad

  • public class ExtensibleGamepad
extends java.lang.Object
    The extensible Gamepad, based on Gamepad. This takes care of multiple issues of the design of the original Gamepad, including code readablity, design, and extending the capilabties of what the Gamepad can do.
    Since:
    0.1
    Author:
    David Sargent
